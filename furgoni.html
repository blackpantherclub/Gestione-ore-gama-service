⚙️ Configura Email Automatiche

Le notifiche vengono inviate automaticamente 1 mese prima, 15 giorni prima e il giorno stesso della scadenza revisione, ogni volta che apri questa pagina.

Come ottenere le credenziali EmailJS (gratis):
1. Vai su emailjs.com → Registrati
2. Email Services → Add Service → Gmail (o altro) → copia il Service ID
3. Email Templates → Create → copia il Template ID
   Nel template usa le variabili: {{targa}} {{modello}} {{scadenza}} {{giorni_mancanti}}
4. Account → API Keys → copia la Public Key
